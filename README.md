# playing-with-code

Welcome to my digital scratchpad! This repository is a public sandbox where I dump random code, test out ideas, and practice programming. 

It is not a single cohesive project, but rather a place for testing algorithms, building small prototypes, actively experimenting, and developing new skills.

## What Could Be Inside?

You could find a mix of different things here as I explore a wide range of computational fields:
* **Machine Learning & Deep Learning:** Exploratory scripts, model architectures, clustering algorithms, and data processing pipelines.
* **Full-Stack Experiments:** Web development components, APIs, and frontend tests.
* **Game Development:** Scripts, blueprints, and project files from Godot and Unreal Engine.
* **Language Practice:** Snippets, algorithms, and sandbox files written in Python, Java, C, C++, and Assembly.
